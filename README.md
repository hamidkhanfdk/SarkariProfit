# 🚀 SarkariProfit

SarkariProfit is a full-stack web application that provides the latest updates on government jobs, schemes, admit cards, results, and useful online tools.

---

## 📌 Features

- 📢 Latest Govt Jobs Updates
- 🏛 Sarkari Yojna Information
- 🪪 Admit Cards & Results
- 🛠 Tools (Image to PDF, Compressor, Photo Maker)

---

## 🛠 Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Spring Boot (Java)
- **Database:** MySQL

---

## 🚧 Project Status

Currently under development — UI is being built and backend integration will be added soon.
